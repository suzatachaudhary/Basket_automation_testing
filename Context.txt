ALERTS & POPUPS

1. JS Alerts (Browser Alerts)
   - Not in DOM → cannot use CSS or XPath
   - Must use:
       driver.switch_to.alert
       alert.accept()
       alert.dismiss()
       alert.text
   - Types:
       alert()
       confirm()
       prompt()

2. HTML Based Popups/Modals
   - Part of DOM → can be inspected
   - Can use CSS/XPath:
       div.modal
       button.close
   - Interaction same as normal elements
   - Usually designed with Bootstrap/JS frameworks

iframe and window Handling
-->An iframe is an HTML element that embeds another webpage inside the current webpage, creating a separate document that Selenium must switch into before interacting with its elements.
driver.switch_to.frame("iframe")

-->for window we have 2 methods current_window_handle and window_handles
and we can switch to next window by using driver.switch_to.window(win)
